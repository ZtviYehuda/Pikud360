# Command Palette Component Specification

**Code Location:** `frontend/src/components/ui/command.tsx`  
**Spec Status:** ✅ Built & Verified  

---

## 1. Purpose
The Command Palette provides global navigation and action execution via `Ctrl+K` keyboard shortcut.

---

## 2. Capabilities
- Keyboard-only navigation (`Up`/`Down` arrows, `Enter` to select, `Esc` to close).
- Search across pages, employees, reports, commands, and quick actions.
- Recent searches & favorite commands history tracking.
- RTL layout support with Hebrew labels.
